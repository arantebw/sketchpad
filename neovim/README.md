# neovim

Reloads the currently open Lua file.

```
:luafile %
```

## Packer

Opens the available Packer-related options.

```
:Packer + <TAB>
```

## Mason

Opens the available Mason-related options.

```
:Mason + <TAB>
```

Opens the Mason modal of available LSPs, DAPs, linters, and formatters.

```
:Mason
```
