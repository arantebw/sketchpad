# kitty

Reload the updated configuration settings.

```text
# linux
ctrl + shift + f5

# macos
control + command + ,
```

Rename an open tab.

```text
Ctrl + Shift + Alt + t
```

Switch between open tabs.

```text
Ctrl + Shift + Left / Right
```

Create a new tab.

```text
Ctrl + Shift + t
```

Zoom in or out the text size.

```text
# Zoom in
Ctrl + Shift + =

# Zoom out
Ctrl + Shift + -
```

Enable full screen mode.

```text
Ctrl + Shift + F11
```

Open the `kitty` configuration file.

```text
# linux
Ctrl + Shift + F2

# macos
command + ,
```
