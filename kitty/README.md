# kitty

Navigate through open containers.

```text
Mod + j/k/l/;
```

Change container layout to tabbed.

```text
Mod + w
```

Zoom in or out the text size.

```text
# Zoom in
Ctrl + Shift + =

# Zoom out
Ctrl + Shift + -
```

Enable full screen mode.

```text
Ctrl + Shift + F11
```

Open the `kitty` configuration file.

```text
Ctrl + Shift + F2
```
