### CLI Commands

#### prettier

Format all of the JavaScript files in the current directory.

```bash
$ prettier --write "**/*.js"
```

#### find

Find a `"filename"` directory at `<location>` directory.

```bash
$ find <location> -type d -name "filename"
```
