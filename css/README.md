# CSS

## <span id="sr-only">CSS `sr-only` Class</span>

```css
.sr-only {
   border: 0;
   clip: rect(1px, 1px, 1px, 1px);
   clip-path: inset(50%);
   -webkit-clip-path: inset(50%);
   width: 1px;
   height: 1px;
   overflow: hidden;
   white-space: nowrap;
   position: absolute;
   padding: 0;
   margin: -1px;
}
```
