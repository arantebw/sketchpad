# MacOS

Create a selected area screenshot.

```text
command + shift + 4
```

Create a fullscreen screenshot.

```text
command + shift + 3
```

Full screen mode the current window.

```text
control + command + f
```

Logout of the current session.

```text
shift + command + q
```

Resize the terminal font sizes.

```text
command + shift + `+`
command + `-`
command + 0
```

Force quit the current open app.

```text
command + q
```

Fill the screen size of an open app.

```text
fn + control + f
```

Lock the screen.

```text
control + command + q
```
